Description
-----------

Prebuilt versions of various things to make it easier to build Grit and related projects on MSVC.

Currently included
------------------

Static libraries dynamic linking (i.e. redistributed DLLs are also needed, not provided here):
* ICU
* Cg
* OpenAL

For static linking:
* freetype
* ogg
* vorbis
* vorbisfile
